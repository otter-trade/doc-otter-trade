# 1.coin-candle 市场模块 使用说明

仓库 Github 地址: <https://github.com/otter-trade/coin-candle>

要解决的问题：

1. 国内获取数据需要`科学上网`
2. 交易所请求规范和行为不统一，解析方式也不统一
3. 回测时对 K 线的消耗比较大，但是频繁去交易所获取效率太慢
4. 需要用到 K 线模块的服务太多了

> 其实最早团队内小伙伴写了一个版本，但我觉得效率不太够，于是自己一个周末肝了出来。
